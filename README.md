# Mon Projet Web

Un site web avec plusieurs pages HTML, styles CSS et ressources multimédia.

## 📁 Structure du Projet

```
projet/
├── html/               # Pages HTML principales
│   ├── accueil.html
│   ├── galerie.html
│   ├── reservation.html
│   ├── patrimoine.html
│   └── ...autres pages
├── css/                # Fichiers de style CSS
│   ├── style.css
│   ├── accueil.css
│   └── ...autres styles
└── assets/
    └── images/        # Images et ressources multimédia
```

## 🚀 Comment utiliser

1. Cloner le repository
2. Ouvrir `html/accueil.html` dans votre navigateur
3. Ou héberger sur GitHub Pages pour y accéder en ligne

- Accueil
- Galeries

- Réservation
- Et d'autres pages explorables
